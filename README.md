# Galatasaray Taraftar Uygulaması ve Kadro Oluşturucu

Bu proje, taraftarların güncel takım kadrosunu inceleyebileceği, oyuncu istatistiklerine erişebileceği ve kendi "Favori 11"lerini oluşturup kaydedebileceği dinamik bir web platformudur.

### Kullanılan Teknolojiler
* PHP (Oturum yönetimi ve veri işleme)
* HTML & CSS (Responsive arayüz tasarımı)
* JSON/Array Manipülasyonu
